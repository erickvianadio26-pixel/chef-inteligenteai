# Chef Caseiro Recipes 

Este projeto é uma aplicação web que utiliza Inteligência Artificial para gerar e gerir receitas personalizadas. Ele foi evoluído a partir de um protótipo feito no Google Stitch para um produto completo e funcional no Lovable.

##  Variáveis de Ambiente Necessárias
Para que o projeto funcione corretamente com o banco de dados e com a IA, configuramos as seguintes variáveis no backend do Supabase:
* VITE_SUPABASE_URL (Link de conexão com o Supabase)
* VITE_SUPABASE_ANON_KEY (Chave de acesso segura)

##  Como rodar o projeto localmente
Caso o avaliador queira testar o código na sua própria máquina, basta seguir estes comandos no terminal:

1. Instalar as dependências:
   npm install

2. Iniciar o servidor de desenvolvimento:
   npm run dev
